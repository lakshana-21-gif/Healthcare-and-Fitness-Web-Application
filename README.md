# Healthcare and Fitness App

A full-stack Health and Fitness Tracking Application developed to help users monitor and manage their health parameters through an interactive dashboard. The application provides real-time tracking, visualization, and analysis of various health metrics.

## Features

- BMI Calculator
- Water Intake Tracking
- Workout Monitoring
- Blood Pressure Tracking
- Blood Glucose Monitoring
- Sleep Analysis
- Daily Step Counter
- Mood Tracking
- Age Index Calculation
- Antioxidant Index Monitoring
- Vascular Health Assessment
- Interactive Charts and Dashboards
- Real-Time Data Updates
- User Authentication and Data Management

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript
- Chart.js

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Deployment
- Docker
- Docker Compose

## Project Architecture

The application follows a full-stack architecture:

- Frontend for user interaction and visualization
- Backend APIs for data processing and management
- MongoDB database for storing health records
- Docker containers for deployment and scalability
